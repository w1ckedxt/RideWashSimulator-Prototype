# Sessie 11 juni 2026 — Ride Wash Simulator (one-shot → v3 prototype)

## Gedaan deze sessie (chronologisch, op feedback van Thomas)
- v1: complete game from scratch (Three.js CDN→lokaal, ES modules, geen bundler);
  Steel Comet met realistische banking, DirtSystem, washer, FPS-controls, NL UI.
- v1.1: lifthill-walkway, ~120 bomen, wolken, plaza, hekken.
- v2: rebrand Ride Wash Simulator (grimey thema), Engelse UI, 5 levels
  (carousel → ship → top spin → woodie → steel) met levelmenu.
- v3 (vele polish-rondes):
  - Geaard poppetje: zwaartekracht, springen, V=hoogwerker; stabiel
    grond-plakken (gesmoothde op/afstapjes, echte rand-detectie);
    onzichtbaar loopdek over beide coasterbanen (niet meer doorvallen).
  - ESC-pauzemenu (resume/restart/menu + settings: geluid, gevoeligheid,
    progress-reset); free play (alles unlocked, prototype).
  - PowerWash-grade vuil: max dekking, zwart↔bruin↔groen variatie, drips,
    zachte smoothstep-randen, streek-interpolatie (geen stippellijn).
  - Per-level: carouselpaarden (5 vachten, manen/zadels), gestreepte luifel,
    goud-decor ook vies; schip-drakenkop + gunwale + vlonders; Top Spin naar
    referentiefoto (rechtopstaande gondel in startpositie, sterren-torens,
    stippenbanner, contragewicht, hoofdas); treinen in beide stations;
    stations met trapjes + airgates; woodie-catwalk.
  - Elke ride: inloopbare operator-booth (deur, doorkijkramen, paneel met
    knoppen/hendel), queue, vies naambord (sectie), bergen, hires wolken.
  - Dag/nachtcyclus gebouwd en daarna op verzoek uitgezet (vast ochtendlicht;
    code blijft aanwezig om te heractiveren).
  - Branding: PROTOTYPE-badge, credits Thomas Geelens (x.com/ThomasGeelens) +
    Lifthill Studio (lifthill.studio), feedback-CTA (X/Reddit/
    Hello@Lifthill.studio) op start- en winscherm. Logo-slot assets/logo.png.
  - three.js + addons lokaal gevendord; GLB-asset-pijplijn klaar (models.js);
    UI groepeert secties per label.
