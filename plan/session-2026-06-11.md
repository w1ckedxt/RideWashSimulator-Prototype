# Sessie 11 juni 2026 — One-shot build Ride Cleaner Simulator

## Gedaan deze sessie
- Volledige game from scratch (greenfield): Three.js 0.170 via CDN, vanilla ES
  modules, geen bundler. Speelbaar via statische server.
- Track-math: gesloten centripetal Catmull-Rom, banking via gevoelde verticaal
  (g + centripetale kracht uit energiebehoud-snelheidsprofiel), ~441 m layout:
  station → lift (26 m) → drop → turnaround → airtime → helix (510°) → remstraat.
- DirtSystem: 6 maskers (rails L/R, ruggengraat, dwarsbalken, steunen, perron)
  als RGBA DataTextures; anisotroop elliptisch gummen, cell/wrap-logica,
  incrementele progressie, secties auto-klaar bij 98,5%.
- Washer: raycast-spray (5 rays/kegel), straal + mist particles, procedureel
  pistoolmodel, afstands-falloff.
- Speler: pointer lock FPS + vrij omhoog/omlaag (hoogwerker). Vuilzoeker (F).
- UI (NL): voortgang totaal + per sectie, toasts, win-scherm, pauze.
- Audio: procedurele spray-ruis, wind, sectie-ding, win-melodie.
- v1.1 op verzoek Thomas ("nog veel beter, echter"): lifthill-walkway met
  dubbele handrail + liftketting, ~120 bomen, wolken, plaza + wachtrij,
  parkhek, perronhek.
- Zelf geverifieerd in Chrome (devtools MCP): 60 FPS, geen console errors,
  erase/progressie/secties werken (debug-hook `window.__game`).
